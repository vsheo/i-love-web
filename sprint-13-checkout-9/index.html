leervragen

  • Wat zijn sveltekit layouts en hoe kan je die toepassen in een web project?
    • dit zijn layouts voot HTMNL CSS en JS die op alle child pagina gebruikt worden
  • Wat zijn componenten en hoe kan je die toepassen in een web project?
    • dit zijn stukken code (HTML, CSS en JS als dat nodig is) die ge importeerd kunnen worden op de plekken waar dat nodig is
  • Kies een van de principes: file-based routing, load functions, runes (reactivity hack) of forms en actions. Leg uit hoe je het gekozen principe kunt toepassen in een web project.
    • file-based routing: de mappen worden door sveltekit gemaakt tot route/URL op de website
    • load functions: functies om bijvoorbeeld data op te halen
    • runes: een functie van sveltekit om variablen te manipuleren, importeren,...
