Leervragen

Zet de antwoorden in je learning journal.

Hoe bereid je een degelijke feedbacksessie voor?

  • in de workshop heb ik deze vragen meegekregen:
    • Wie is de doelgroep voor dit product?
    • Welk gebruikersprobleem lost dit product voor hen op?
    • Wat is de kernwaarde voor gebruikers?

deze vragen laat je goed nadenken voor wie de website bedoeld is en wat je als gebruiker op de website zou willen zien/doen.
deze vragen zal ik meenemen naar de volgende code en design review

Hoe geef je jouw feedback?

  • ik probeer zo objectief als mogelijk te reageren. ik geef tips als iets beter gechreven kan worden, bijvoorbeeld om duplicate code te voorkomen.
    en ik link naar hulpbronnen als dat nodig is.

Hoe verwerk je ontvangen feedback?

  • ik probeer eerst te begrijpen als wat de feedback is, en stel vragen als ik iets niet zeker weet of niet zo goed begrijp.
    daarna ga ik de feedback verwerken. in de issue bedank ik de persoon voor de feedback en meld ik ook als het feedback verwerkt is.
